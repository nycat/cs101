# MySQL

## Data type

- time
  - timestamp

    - The `TIMESTAMP` value has a range from '1970-01-01 00:00:01' UTC to '2038-01-19 03:14:07' UTC.
    - The format of a TIMESTAMP column is YYYY-MM-DD HH:MM:SS which is fixed at 19 characters.

  - datetime
